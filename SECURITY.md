# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

* Email: [security@orqestly.dev](mailto:security@orqestly.dev)
* Do not open public issues for security vulnerabilities

## What to Include

* Description of the issue
* Steps to reproduce
* Impact assessment

## Response Time

We aim to respond within 72 hours.

## Scope

This includes:

* Core engine
* Agents
* Integrations

Thank you for helping keep Orqestly secure.
